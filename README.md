## À propos du projet

Pokemon est un Pokedex des 30 premiers Pokemon d la première génération. Venez les redécouvrir et parcourir toutes leurs spécificités.

C'est un projet qui se présente en deux temps:
+ Récupérer les Pokémons depuis une API et les afficher.
+ Afficher les spécificités d'un Pokemon en particulier.

Le tout en MVC en PHP Vanilla.

### Technologie utilisée
+ HTML
+ CSS
+ PHP
